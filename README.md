# 🎓 UniEvents -- University Event Management System

UniEvents is a **web-based University Event Management System** that
allows students and organizers to manage campus events efficiently. The
platform enables users to browse events, register for them, and track
their registrations. Event organizers can create and manage events
through an admin dashboard.

This project demonstrates the use of **Fundamentals of Web Development
(HTML, CSS, JavaScript)** along with **Data Structures and Algorithms
(DSA)**.

# 📊 Dashboard

After logging in, users access the **dashboard**, which includes:

### 📈 Overview

Displays: - Total events - Open registrations - User registrations -
Upcoming events - Recently added events

### 🔎 Browse Events

Users can: - Search events by **title or venue** - Filter events by
**category** - View event details - Register for events

### ✅ My Registrations

Users can: - View all registered events - Cancel registrations

------------------------------------------------------------------------

# 🛠 Event Management (Admin Only)

### ➕ Add Event

Admins can create events by providing: - Event title - Category -
Venue - Date and time - Total seats - Registration deadline -
Description

### 📋 Manage Events

Admins can: - View all events - See the number of registered
participants - Delete events

------------------------------------------------------------------------

# 🧠 Data Structures and Algorithms Used

### 1️⃣ Linear Search -- O(n)

Used to find a user by email during login.

### 2️⃣ Binary Search -- O(log n)

Used to efficiently locate events by event ID.

### 3️⃣ Queue (FIFO)

Used to process event registrations in **First-In-First-Out order**.
